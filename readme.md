# Sketch CAD

Sketchcad is an application that helps you to easily sketch CAD like drawings or markup existing images with cad symbols.

It works very much like draw.io or drawing shaps in powerpoint so the learning curve is minimal.

It features common cad elements such as:
- infinite horizontal lines
- infinate vertical lines
- line segments
- circles
- arcs
- measurements
- points
- annotations (text)
- arrows
- view symbols (like A-A view)
- rectangles
- raster images

All these elements are vector objects and can be manipulated by selecting and then dragging their handles.

Snapping is build-in for easy alignment

Measurements do not measure themselves, we just need to manually put in the number.

It is a sketching program, not a cad program. There is no scale and there is no relation between the drawing scale and displayed measurements. It is really just a drawing program with cad symbols.

## File format

Files can be saved to svg or pdf.

Files can be saved to and opened from a -to be determined- format.

## Style

Simple, minimal and modern style. Light. Simple menu bar on top and palette on the left.


## Framework

Sketchcad runs on Tauri + TypeScript
